Renta Fija: calculo de devengo entre cartolas

Mismo principio que dividendos — solo contamos lo que pasa entre snapshots

Ejemplo: Santander 5.294% semi-annual, $50,000 nominal

Snapshot A (31-Mar) ────────────────────── Snapshot B (30-Abr)
Devengo del periodo:
cuponSemestral = 5.294% / 2 = 2.647%
cuponDiario = 2.647% / 180 dias (30/360) = 0.01471%
diasEntrCartolas = 30 dias
devengo = $50,000 x 0.01471% x 30 = $220.58
Diferencia de precio:
precioA = 98.50 → precioB = 99.12
difPrecio = (99.12 - 98.50) / 100 x $50,000 = +$310
Cupon cobrado en el periodo:
Fecha cupon: 15-Feb (ya paso) / 15-Ago (no ha llegado)
Cupones cobrados entre 31-Mar y 30-Abr: $0
Retorno total periodo = devengo + difPrecio + cupones
= $220.58 + $310 + $0 = $530.58 (+1.06%)

Pregunta: como detectamos cupones cobrados entre cartolas?

Los bonos pagan cupon semi-annual. Si cae entre dos snapshots, hay que contarlo.

Datos que ya tenemos por bono

couponRate: 5.294%
maturityDate: 2027-08-15
couponFrequency: 2 (semi-annual)
faceValue / nominal: $50,000
Con maturityDate + frequency podemos calcular las fechas de cupon hacia atras
A

Calculo automatico desde maturity + frequency

Ya tenemos cash-flows.ts que genera el schedule de cupones. Filtramos cuales caen entre fecha snapshot A y B. Monto = nominal x cuponRate / frequency.

Pros

  • 100% automatico, ya tenemos la logica
  • No requiere input del asesor

Cons

  • Asume bullet bond con cupones regulares
  • Fecha exacta puede tener +/- 1-2 dias por feriados
B

Ingreso manual del asesor

Boton "Registrar cupon" por bono, similar al registro de dividendos que ya existe. El asesor ingresa fecha y monto.

Pros

  • Preciso: monto real recibido
  • Captura withholding tax si aplica

Cons

  • Mas trabajo para el asesor
C

Auto-calculo con override manual

Se calcula automaticamente desde el schedule. Si el asesor quiere corregir (ej: retencion de impuesto, fecha ajustada), puede editar el monto.

Pros

  • Funciona sin input, pero permite precision
  • Best of both worlds

Cons

  • Mas UI para el override
